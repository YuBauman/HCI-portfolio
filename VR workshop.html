Advanced: 05 Interactions

05 Interactions: Step 1-4
In this block I learned how to trigger the animations based on the events like mouseenter and mouseleave. I added the rotation speed to the box as the mouse enters the box, or stretched the width of the box. In previous exercises, as the mouse hovered over the box, it kept rotating at the same speed always after. According to the assignment, I changed the code the way that once the mouse leaves the box, its speed gets back to the original one.

Advanced: 06 Interaction Headset

06 Interaction Headset: Step 3
In this step I customised the cursor apperance, so I expanded the radius of the circle that represents the cursor, and added the color animation, so cursor color fades from blue to pink. After that I was playing around with the fuse function that delays the outcome once the cursor is over the element. I changed the fuseTimeOut to 1s instead of 1.5s, so it reacts faster, and made the rotation speed of the right box increase every time the cursor is over the box. The left box changes its rotation by 45 degrees on all axis every time the cursor is over the right box.
